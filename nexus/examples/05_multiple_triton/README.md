# Example 05: Multiple Triton Kernels Tracing

Example showing how to use Nexus to trace multiple Triton kernels in a single application.

## Run it

```bash
python3 multiple_triton.py
```

## Requirements

- ROCm installed
- Triton installed: `pip install triton`
- Nexus installed: `pip install git+https://github.com/AMDResearch/nexus.git#egg=intellikit[nexus]`

See [nexus documentation](../../README.md) for more details.
