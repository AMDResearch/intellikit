## What's in the box

IntelliKit is a set of Python tools for AMD-focused performance and validation. The stack targets GPUs through ROCm, turning hardware counters, traces, and dispatch data into clean APIs you can use from Python, scripts, MCP clients, or AI coding assistants.

<CardGrid>
  <Card title="Kerncap" icon="rocket">
    **Isolate** — capture kernel dispatches and build standalone reproducers for HIP and Triton applications.
  </Card>
  <Card title="Metrix" icon="bars">
    **Profile** — human-readable metrics from hardware counters: bandwidth, cache, coalescing, compute.
  </Card>
  <Card title="Linex" icon="magnifier">
    **Profile** — source-line timing and stall analysis. Map GPU performance to your code.
  </Card>
  <Card title="Nexus" icon="puzzle">
    **Inspect** — intercept HSA packets to see what ran on the GPU: assembly and HIP source.
  </Card>
  <Card title="Accordo" icon="approve-check-circle">
    **Validate** — prove an optimized kernel still matches a reference implementation.
  </Card>
  <Card title="ROCm MCP" icon="setting">
    **MCP** — HIP compiler, HIP docs, and rocminfo servers for LLM agents.
  </Card>
  <Card title="uProf MCP" icon="laptop">
    **CPU** — MCP bridge to AMD uProf for host-side CPU hotspot analysis.
  </Card>
</CardGrid>

## The workflow

**Isolate** a kernel with Kerncap, **profile** it with Metrix and Linex, **inspect** execution with Nexus, wire agents via **MCP servers** and **skills**, add **uProf** for host-side analysis, then lock in correctness with **Accordo**.

```python
from metrix import Metrix
from nexus import Nexus
from accordo import Accordo

# 1) Profile — human-readable GPU metrics
profiler = Metrix()
baseline = profiler.profile("./app", metrics=["memory.hbm_bandwidth_utilization"])

# 2) Inspect — see what ran on the GPU
trace = Nexus().run(["./app"])
for kernel in trace:
    print(kernel.name, len(kernel.assembly), "instructions")

# 3) Validate — check correctness after optimization
validator = Accordo(binary="./app", kernel_name="my_kernel")
ref = validator.capture_snapshot(binary="./app_ref")
opt = validator.capture_snapshot(binary="./app_opt")
result = validator.compare_snapshots(ref, opt, tolerance=1e-6)
print(f"{'PASS' if result.is_valid else 'FAIL'}")
```
